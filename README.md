# Overview

Tool for updating resumes

## Setup

### Requirements

* Python 3.10+

### Installation

Install it directly into an activated virtual environment:

```text
$ pip install resumai
```

or add it to your [Poetry](https://poetry.eustace.io/) project:

```text
$ poetry add resumai
```

## Usage

After installation, the package can be imported:

```text
$ python
>>> import resumai
>>> resumai.__version__
```
